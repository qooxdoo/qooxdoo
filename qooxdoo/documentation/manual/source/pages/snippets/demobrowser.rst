Demo Browser
************
The **Demo Browser** has a large number of simple examples. You can use the `online version <http://demo.qooxdoo.org/%{version}/demobrowser>`_ or if you've downloaded and :doc:`built the SDK </pages/tool/generator_usage>`, you can use your local copy in ``application/demobrowser/build`` subdirectory of your SDK installation.
