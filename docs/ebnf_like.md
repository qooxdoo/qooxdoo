# Syntax Diagrams

A short summary of the elements used in syntax diagrams.

| Syntax symbol    | Description                                               |             |
| ---------------- | --------------------------------------------------------- | ----------- |
| **:=**           | production rule (_"non-terminal can be expanded to ..."_) |             |
| **{** ... **}**  | 0..N occurrences (_"\*"_)                                 |             |
| **\[** ... **]** | 0..1 occurrences (_"?"_)                                  |             |
| **(** ... **)**  | grouping                                                  |             |
| ... \*\*         | \*\* ...                                                  | alternative |
| **'** ... **'**  | literal                                                   |             |
| **(** ... **)**  | placeholder for literal                                   |             |
| **?** ... **?**  | comment                                                   |             |
