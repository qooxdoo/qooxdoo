Annotations

  • Mobile and desktop devices should be connected to "onair" network.
  • On iOS you only need to enter credentials. For deletion of your credentials, just disconnect once.
  • On Android you need to enter credentials (identity and password), chose EAP method >> TTLS, and Phase2 authentication >> PAP. After testing on Android, please delete WLAN connection.
  • [a] Tablet view has 2 NavigationPages on a screen on landscape orientation, and 1 NavigationPage + Popup in portrait mode.
  • [b] Mobile view just has one NavigationPage and a "Back" button at NavigationBar.
  • [c] Orientation change can be reached either, by rotating the tablet device by 90 degrees, or resizing the browser windows size to less/above 50:50 ratio
